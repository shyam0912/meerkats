# 🚀 Meerkats - Project Progress

## Project Vision

Meerkats is an Interactive Panel Software designed for schools, enabling teachers to conduct engaging digital classroom sessions with whiteboard tools, subject-specific teaching content, AI assistance, simulations, and classroom management.
